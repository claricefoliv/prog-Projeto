Este projeto tem o intuito de reproduzir um programa no modelo Paint em POO, sendo notável a evolução de códigos imperativos até o paradigma orientado a objetos.

## 🚀 O objetivo de aprendizado é:
- Modelar um domínio com uma hierarquia de classes e polimorfismo (Orientação a Objetos).
- Aplicar a arquitetura MVC e os padrões de projeto State, Composite e Command.
- Persistir e recuperar o estado da aplicação (serialização / Salvar e Abrir).
- Escrever testes unitários para a camada de modelo (ver Entrega 4).
- Trabalhar de forma colaborativa com Git e Github (branches, commits, tags, divisão de tarefas).

## 🛠️ Pré-requisitos
Antes de executar o projeto, certifique-se de ter o Python 3 instalado em seu dispositivo. A biblioteca gráfica utilizada (Tkinter) já vem na instalação padrão do Python na maioria dos sistemas operacionais!
